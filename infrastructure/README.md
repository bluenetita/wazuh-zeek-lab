# Infrastructure Systems

This directory documents the systems monitored or used as targets in the laboratory.

| Path | Role |
|---|---|
| `client-linux/` | Main Linux endpoint for Auditd, FIM, AppArmor, reverse shell, and privilege escalation |
| `client-windows/` | Additional Windows endpoint |
| `server-db/` | Database/server role |
| `victim-server/` | Additional controlled target |

Each directory should describe role, network placement, telemetry, security controls, and sanitized deployment notes.
