# Wazuh Evidence — Privilege Escalation

Acceptable samples include reduced alerts for rules `100700–100704`, related Auditd events, and relevant FIM context.

Each artifact should identify:

- Auditd key;
- rule ID and level;
- executable and process context;
- UID/AUID/EUID values;
- what the event demonstrates;
- known benign explanations.

Do not publish complete logs, credentials, payloads, exploit code, or personal data.
