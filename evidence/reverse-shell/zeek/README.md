# Zeek Evidence — Reverse Shell

Store small sanitized examples from:

- `conn.log`;
- `possible_malware.log`;
- `reverse_shell_live.log`;
- `reverse_shell_movement.log`;
- `reverse_shell_final.log`.

Each artifact should identify the connection UID and explain its role in the lifecycle. Replace real addresses, timestamps, ports, filenames, and hashes when appropriate.

Do not publish complete logs, PCAPs, payloads, or attacker commands.
