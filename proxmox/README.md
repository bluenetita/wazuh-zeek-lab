# Proxmox Configuration

This directory documents the virtualization and network-observation layer of the laboratory.

| File / path | Purpose |
|---|---|
| `network-bridges.md` | Linux/OVS bridge design |
| `ovs-mirroring.md` | Mirror configuration for Zeek |
| `routing.md` | Host-level routing notes |
| `vm-inventory.md` | VM roles and interfaces |
| `ovs/` | Reproducible OVS scripts |
| `nftables/` | Host firewall notes |

Do not publish cluster credentials, storage secrets, API tokens, VM disks, or complete host backups.
